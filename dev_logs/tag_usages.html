HTML 태그별 사용법

기본 구조 태그

1. <DOCTYPE! html> 태그

<DOCTYPE! html> 태그는 HTML 문서의 버전을 지정하는 데 사용됩니다. 이 태그는 문서의 첫 번째 줄에 위치해야 하며,
HTML5 버전을 사용할 때는 <!DOCTYPE html>로 표기합니다.

2. <html> 태그

<html> 태그는 HTML 문서의 시작과 끝을 표시해 줍니다. 모든 요소는 이 태그 안에 포함되어야 합니다.

3. <head> 태그

<head> 태그는 문서의 메타데이터를 포함하는 데 사용됩니다. 메타데이터란 데이터에 관한 데이터를 의미합니다.

여기에는 제목, 문자 인코딩, 스타일 시트 링크 등이 포함됩니다.

4. <meta> 태그

<meta> 태그는 문서의 메타데이터를 포함하는 데 사용됩니다. 여기에는 문자 인코딩, 설명, 키워드 등이 포함됩니다.

5. <title> 태그

<title> 태그는 브라우저 탭에 표시될 제목을 기입하는 태그입니다. 검색 엔진 결과에도 영향을 미칩니다.

6. <header> 태그

<header> 태그는 문서나 섹션의 헤더를 나타내는 데 사용됩니다. 일반적으로 제목과 함께 사용됩니다.

7. <main> 태그

<main> 태그는 문서의 주요 콘텐츠를 나타내는 데 사용됩니다. 페이지의 핵심 내용을 포함하는 요소입니다.

8. <section> 태그

<section> 태그는 문서의 구획을 나타내는 데 사용됩니다. 관련된 콘텐츠를 그룹화하는 데 유용합니다.

9. <article> 태그

<article> 태그는 독립적인 콘텐츠를 나타내는 데 사용됩니다. 블로그 게시물, 뉴스 기사, 사용자 댓글 등에 사용됩니다.

10. <h1> ~ <h6> 태그

<h1>부터 <h6>까지의 태그는 제목을 나타내는 데 사용됩니다. 숫자가 작을수록 더 중요한 제목을 나타냅니다.

h1부터 h2까지 중요도에 따라 순서대로 사용해야 하며 h1은 한 페이지에 하나만 사용하는 게 좋습니다.

11. <p> 태그

<p> 태그는 단락을 나타내는 데 사용됩니다. 텍스트를 여러 단락으로 나누어 가독성을 높이는 데 유용합니다.

12. <br> 태그

<br> 태그는 줄바꿈을 나타내는 데 사용됩니다. 단락 내에서 줄을 바꿔야 할 때 유용합니다.

예를 들어 브라우저 및 모니터 크기로 인해 단어가 끊어지는 경우 사용하면 좋습니다.

단락을 나누는 데도 사용할 수는 있지만, <p> 태그를 사용하는 것이 더 권장됩니다.

13. <hr> 태그

<hr> 태그는 수평선을 나타내는 데 사용됩니다. 페이지 내에서 구분을 나타내거나 섹션을 나누는 데 유용합니다.

요즘은 글의 분위기 반전을 위해 사용되는 경우가 많습니다.

14. <blockquote> 태그

<blockquote> 태그는 긴 인용문을 나타내는 데 사용됩니다. 다른 사람의 말이나 글을 인용할 때 유용합니다.
