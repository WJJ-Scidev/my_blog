HTML 문서의 기본 구조와 시맨틱 태그

오늘은 HTML 문서의 기본 구조와 시맨틱 태그에 대해 학습했습니다. HTML은 웹 페이지를 구성하는 기본적인 언어로, 시맨틱 태그는 문서의 의미를 명확하게 전달하는 데 사용됩니다.

HTML 문서는 <!DOCTYPE html> 선언으로 시작하며, <html>, <head>, <body> 태그로 구성됩니다. 시맨틱 태그에는 <header>, <nav>, <main>, <section>, <article>, <footer> 등이 있습니다. 이러한 태그들은 웹 페이지의 구조를 명확하게 표현하고, 검색 엔진 최적화(SEO)에도 도움이 됩니다.

오늘 배운 내용을 바탕으로 간단한 HTML 문서를 작성해보았습니다. 앞으로도 계속해서 HTML과 CSS, JavaScript 등 웹 개발에 필요한 기술들을 학습해 나갈 계획입니다.
