1. <DOCTYPE! html> 태그

<DOCTYPE! html> 태그는 HTML 문서의 버전을 지정하는 데 사용한다. 이 태그는 문서의 첫 번째 줄에 위치해야 하며,
HTML5 버전을 사용할 때는 <!DOCTYPE html>로 표기한다.

2. <html> 태그

<html> 태그는 HTML 문서의 시작과 끝을 표시해 준다. 모든 요소는 이 태그 안에 포함되어야 한다.

3. <head> 태그

<head> 태그는 문서의 메타데이터를 포함하는 데 사용된다. 메타데이터란 데이터에 관한 데이터를 의미한다.

여기에는 제목, 문자 인코딩, 스타일 시트 링크 등이 포함됩니다.

4. <meta> 태그

<meta> 태그는 문서의 메타데이터를 포함하는 데 사용된다. 여기에는 문자 인코딩, 설명, 키워드 등이 포함된다.

5. <title> 태그

<title> 태그는 브라우저 탭에 표시될 제목을 기입하는 태그다. 검색 엔진 결과에도 영향을 미친다.

6. <header> 태그

<header> 태그는 문서나 섹션의 헤더를 나타내는 데 사용된다. 일반적으로 제목과 함께 사용된다.

7. <main> 태그

<main> 태그는 문서의 주요 콘텐츠를 나타내는 데 사용된다. 페이지의 핵심 내용을 포함하는 요소다.

8. <section> 태그

<section> 태그는 문서의 구획을 나타내는 데 사용된다. 관련된 콘텐츠를 그룹화하는 데 유용하다.

9. <article> 태그

<article> 태그는 독립적인 콘텐츠를 나타내는 데 사용된다. 블로그 게시물, 뉴스 기사, 사용자 댓글 등에 사용된다.

1~9 사용 예시

<!DOCTYPE html>
<html lang="ko">
<head>
<meta charset="UTF-8">
<title>문서 제목</title>
</head>
<body>
<header><h1>헤더 제목</h1></header>
<main>
<section>
<article><p>독립적인 콘텐츠</p></article>
</section>
</main>
</body>
</html>

출력

헤더 제목

독립적인 콘텐츠

<?>: 닫는 태그가 없는 태그

표는 다음 그림과 같이 표 제목, 열 이름, 본문, 요약 부분으로 나뉜다.

위 그림과 같은 표를 생성하려면 <table> 태그를 사용한다. 표를 만들겠다고 선언하는 과정이다.
그 표에 제목을 추가하려면 <caption> 태그를 사용한다.
일반적으로 엑셀의 경우 행, 열 기반으로 데이터를 기입하지만 html은 행을 기반으로 각 행에다 데이터를 넣으면 열은 알아서 조정되는 방식이다.
예를 들어 2 x 3 표 중 가장 오른쪽 아래 셀이 비어있게 만들고 싶으면 두번째 행에서 세번째 데이터를 안넣으면 된다.
행은 <tr>태그를 넣어 만든다.
데이터는 <td> 태그를 넣어 만든다.
상세 설명은 아래 확인.

1. <table> 태그

<table> 태그는 표를 나타내는 데 사용된다.
일반적으로
<caption>(테이블 캡션(제목))
<tr> (테이블 행, table row)
<th> (테이블 헤더, table header)
<td> (테이블 데이터, table data)
태그와 함께 사용된다.

예시는 <td> 태그 참조

2. <tr> 태그

<tr> 태그는 표에서 행을 나타내는 데 사용된다.
일반적으로 <table> 태그와 함께 사용된다.
<tr> 태그 안에는 <th> 또는 <td> 태그가 포함된다.

예시는 <td> 태그 참조

3. <th> 태그

<th> 태그는 표에서 헤더 셀을 나타내는 데 사용된다.
일반적으로 <tr> 태그 안에 사용되며, 텍스트는 기본적으로 굵게 표시되고 가운데 정렬된다.

예시는 <td> 태그 참조

4. <td> 태그

<td> 태그는 표에서 데이터 셀을 나타내는 데 사용된다.
일반적으로 <tr> 태그 안에 사용되며, 텍스트는 기본적으로 왼쪽 정렬되고 굵게 표시되지 않는다.

코드

<table>
  <tr>
    <th>헤더 1</th>
    <th>헤더 2</th>
  </tr>
  <tr>
    <td>데이터 1</td>
    <td>데이터 2</td>
  </tr>
</table>

데이터 1 데이터 2

일반적으로 <table> 태그 안에 사용되며, <th> 태그로 헤더 행을 포함한다.

<thead> 태그와 마찬가지로 고정시킨 채 본문 부분만 스크롤 할 수 있도록 설정할 수 있다.
